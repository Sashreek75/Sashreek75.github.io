Functions can have ingredients (inputs) ‚Äì these are called parameters.
It's like giving your magic spell something special each time!

Let's make a function that says hello to anyone you want: 
def say_hello(name):
    print("Hello,", name)

How do we use it?
say_hello("Ava") prints Hello, Ava
say_hello("Jake") prints Hello, Jake
The thing inside the parentheses is our magic ingredient!

What does return do?
  • return gives an answer back from your function‚Äîalmost like a magic mailbox!
  • When Python sees return, it stops running the function and brings back the value.
  • You can now *use* the answer anywhere: in math, a new variable, another function, or to show your friend!
Here's a function that returns double a number: 
def double(x):
    return x * 2

This means double(5) "gives you back" 10.
Try printing: 

print(double(5))  # Shows 10
number = double(7)
print(number)     # Shows 14

The function brings back the answer, and you can store it for later!

Comparing print and return:
print() return
What does it do? Shows text on the screen
but doesn't save it		 Gives an answer back, so you can use it in code
Example 
def show(x):
    print(x)
 
def get(x):
    return x
How do you use it? 
show(8)   # prints 8
 
result = get(8)
print(result)   # prints 8
You can use return in lots of ways:
  • Use the result in more math: triple_double = double(3) * 2
  • Save many answers at once! (Python lets you return multiple values)
    def stats(nums):
    return min(nums), max(nums)

lo, hi = stats([3, 6, 2, 8])
print(lo, hi)    # prints 2 8
  • Stop a function early (like a magic escape hatch!): 
    def check(num):
    if num > 5:
        return "Big!"
    return "Small."
What happens if you don't use return?
  • The function finishes, but you get back None.
  • You can't use the answer later‚Äîit's lost!
You can even return lists, dictionaries, or other functions! 
def make_list(x, y):
    return [x, y]

def secret_spell():
    def wow():
        return "Magic!"
    return wow
