Level 2: Variable Vault - Part 2! 🔑

🧠 Did you know? Variables can change their values as your program runs! This is unlike constants, which do the same thing as variables, but cannot be changed! 
 score = 5
print(score)   # shows 5


 score = 10
print(score)   # now shows 10!

See? You can update a variable anytime. It's like swapping out the toy inside your box for a new one! 🎁

💡 Need Help? You might have noticed some errors in your variable names! That's because Variable names can't have spaces, and they can't start with a number. Try names like my_pet or age2! 🐾

You know, you can use a variable and a print function together too!🧩 You can use a variable, and then print out that variable so it shows! Observe:
