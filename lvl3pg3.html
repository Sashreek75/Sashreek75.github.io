üéØ The first thing we went over is what an input is! An input statement is a statement that asks the user a question! Your program will actually stop and wait for the user to answer!

The next thing is all about the type! Remember, unless you use one of the things we talked about, the result will always be a string!

ans = input("what is 1 + 5?:")
      print(ans)

# if the user enters 6, then it'll show up as "6"!

ans = int(input("what is 1 + 5?:"))

# This time, if the user enters 6, (which they should) then it'll show up as 6!
So, as you can see, even if you think 7 and "7" are the same number, the computer thinks they're different!
And finally, we talked about how you can change the data type of an input statement from a string to an integer and float!

      var = float(input("Hello, enter whatever u want!:"))
