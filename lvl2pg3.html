Level 2: Variable Data Types! 🧠

Alright, no more secrets! 🎩✨ I think it's time to tell you guys about something called Data Types! In Python, variables can hold different types of data. Here are four common data types you'll use:

1. Integer (int) 🔢

Integers are just whole numbers without decimals! Lets check out some examples: 
5
-10
0
Pretty simple right? 😊 Notice: Integers can also be negative numbers!

2. String (str) 💬

The second data type we've seen is called a string! This is just words wrapped around with those quotes! See those strings below? ✍️

"Hello, world!"
'Python is fun'
Ok, so you've seen the first two data types in the first level, but now, lets look at something new! Booleans! 🎯

3. Boolean (bool) ✅❌

Booleans are pretty cool because the only values a boolean has are true and false! 😉

Think of it like a light switch. It can only be two things:
✅ ON (which means “Yes” or True)
❌ OFF (which means “No” or False) 
True
False
In the end, a boolean is just a fancy way for a computer to say yes or no!

4. Float (float) 🌊

Now, our last data type is a float! It's very similar to an integer, in fact, I want you to try and guess the difference! 🤔

3.14
-0.5
2.0

Did you get it? All floats have decimal points! It doesn't matter if the decimal is a 0, as long as it has a point, it's a float! 💡

Now, I'm sure you're a pro at data types now, but just in case,you're a little stuck, Python has a tool to help you out! 🛠️
You can check the type of any variable or value using the type() function in Python! 🕵️

Example:

x = 10
print(type(x))  # <class 'int'>

name = "Alice"
print(type(name))  # <class 'str'>

is_happy = True
print(type(is_happy))  # <class 'bool'>

pi = 3.14
print(type(pi))  # <class 'float'>
