Some loops in Python keep going while something is true. That's called a
while loop!
It's like playing a game:
While you're not tired, you keep jumping! 

count = 0
while count < 5:
  print("Jump!")
  count = count + 1

This while loop prints "Jump!" 5 times.
The loop keeps going as long as count < 5 is true.
That's called a conditional! Don't forget to update count inside the loop, or it will never end!

Let's look closely at this line: count = count + 1

This is called incrementing the counter. It means we add 1 to the current value of count every time the loop runs.

Why do we do this?
By increasing count each time, we get closer and closer to making the loop's test count < 5 become False.
Once count is 5, the loop stops, because 5 < 5 is not true!

If we forget this line, the number NEVER changes and the loop would go forever! 

count = 0
while count < 5:
    print("Jump!")
    count = count + 1

Step by step, here's what happens:
1. Start at count = 0
2. Is count < 5? Yes! Print "Jump!"
3. Add 1 (count = 1)
4. Is count < 5? Yes! Print "Jump!"
5. And so on... until count = 5
6. Now count < 5? is not true‚Äîso the loop stops!
Final Output: 

Jump!
Jump!
Jump!
Jump!
Jump!

Pro Tip: The line count = count + 1 is what helps your loop "move forward" and eventually stop at the right time!
Did you know? A while loop can go forever if you don't change the thing it's checking (this is called an infinite loop).
